**【三件事情】**

輸入（Input）、處理（Process）及輸出（Output）

**【五大單元】**

中央處理單元（CPU）［算術邏輯單元（ALU）及控制單元（CU）］、輸入單元（IU）、記憶單元（MU）、輸出單元OU

**◇ 輸入單元（IU）**

是將資料或程式輸入，並轉換為二進位傳送到記憶單元中。就像是在神經系統傳遞中的「**受器**」的位置，負責接收訊息。而常見的輸入單元有滑鼠、鍵盤、繪圖板、掃描器、麥克風、光碟機、照相機等。

**◇ 輸出單元（OU）**

是將程式或資料經過算數及邏輯運算後，轉換成使用者看得懂的文字、圖像、音訊、視訊，再顯示出來。如同在神經系統中，扮演「**動器**」的角色，將指令表現出來。而常見的輸出單元有螢幕、印表機、喇叭等。

**◇ 記憶單元（MU）**

是用來儲存程式或資料的地方。好比是在神經系統中，扮演**「脊髓」(??海馬迴吧)** 的角色。

而記憶單元又分為兩類－主記憶體、輔助記憶體。主記憶體：電腦內部的記憶體，存放輸入的程式或資料，以便運算或輸出，如：RAM。輔助記憶體：電腦外部的記憶體，是為了彌補主記憶體空間不足，如：硬碟、光碟等。

**◇ 控制單元（CU）**

會從主記憶體擷取指令碼再加以解碼，發出控制信號，並指揮及監督每一個單元，以確保電腦正常運作。就像是在神經系統中的「**大腦**」的角色，負責發出指令。()

**◇ 算數邏輯單元（ALU）**

是負責將記憶單元送來的程式或資料，進行「算術」及「邏輯運算」之後，再將結果送回「記憶單元」儲存起來。

其中：

-算術運算：進行四則運算

-邏輯運算：先進行比較運算（大於 / 等於 / 小於），再做邏輯運算（邏輯閘：AND、OR、NOT）

而算數邏輯單元內以加法器來做運算，並用暫存器來存放運算過程中的資料或程式，**與控制單元一同擔任「大腦」的職位**。

-----

\1. 使用者透過「**輸入單元**」輸入資料或程式到「**記憶單元**」

\2.「**控制單元**」從記憶單元中讀取資料或程式，控制動作皆由控制單元處理

\3. 如果要進行運算，必須經過「**算術邏輯單元**」來進行算術或邏輯運算

\4. 處理完後再傳送給「**記憶單元**」

\5. 最後將結果送到「**輸出單元**」來完成動作

-----

**GPU** 為 Graphics Processing Unit 的縮寫，中文是圖形處理器，又稱為顯示晶片，專門在計算機上執行繪圖運算工作，所以它其實是常聽到的「顯示卡」中的核心零件。（顯示卡是 GPU + 記憶體 + 電路板 + 散熱器等的統稱）也可以直接內嵌入主機板或 CPU 上。
